What is this project?

This is a simple Multi-Page Application (MPA) demonstration. Unlike Single-Page Applications (SPAs), each navigation action loads a completely new HTML document from the server.

Key Characteristics

  • Each page is a separate HTML file
  • Full page reloads on navigation
  • Simple server-side routing
  • SEO-friendly by default
  • Easy to understand and maintain

Technology Stack

  • HTML5 for structure
  • CSS3 for styling
  • Vanilla JavaScript (when needed)
